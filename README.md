## 📌 About the Project

This C++ console application allows the user to select a movie from a list.  
The program checks the user's age and restricts access to certain content.

---

## 🚀 Features

- 🎥 Movie selection system
- 🔞 Age restriction validation
- 🔁 Use of functions
- 📊 Array handling
- ⚠️ Input validation

---

## 🛠️ Technologies

- C++
- Standard Library (`iostream`, `string`)

---

## 💻 How it works

1. User enters age
2. Chooses a movie
3. Program validates access
4. Displays message
